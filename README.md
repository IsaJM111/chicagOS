# chicagoOS
An OS that is Chicago themed, complete with a few bassic apps including Notes, Calculator, and Gallery.

**Demo link ->** https://isajm111.github.io/chicagOS/

# Features
- Date and local time, updates every second
- Notes app
  - Add new notes
  - Multiple notes open at once
  - Notes save after app is closed
- Calcuator app
  - Standard 4 function calculator
  - Delete button, remove typos without clearing the whole field.
- Gallery app
  - View preset images
- All windows dragable and closable
